清除浮动

最简单的清除

CSS

        
            overflow: hidden; /*给浮动的父元素添加*/
外面灰色盒子不设置高度，正常情况下会被蓝色盒子撑开高度



常用的伪元素清除

CSS

        
            .container2:after{ /* 给浮动元素的父元素的伪元素添加样式*/
                content: '';
                display: block;
                clear: both;
            }
外面灰色盒子不设置高度，正常情况下会被蓝色盒子撑开高度
